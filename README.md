# タイトル
　shooting game
# 概要
　遮蔽物を利用しながら敵を撃破していくリアルタイムシューティングゲーム。
  武器種やバーストスキルを駆使して強敵との戦闘に挑む。
# 操作方法
  マウスでエイム
  左クリックで射撃
